== GH Ad Process ==

1. prepare PSD
	1. Rename folders "frame#" where # is it's index
	2. Rename layers (ex: "t#" for text where # is it's index)
	3. Select layers in folder, right click and select "Quick Export to PNG"
	4. run `banner-rename.sh <file prefix> <frame number>`
2. prepare files

3. timing
	1. ads should be 15 seconds long, max
	2. css transform/translate/opacity is faster than animating padding/margin/top/left
